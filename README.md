# dotfiles

Collection of config files.

## Requirements

### Ghostty terminal

Install Ghostty terminal.

### GNU Stow

1. Install `stow` - `brew install stow`
2. Checkout to `~/dotfiles`
3. Stow required configs:

```bash
cd ~/dotfiles
stow nvim
```

### Fonts

Install Nerd Fonts

- `ttf-meslo-nerd`
- `ttf-jetbrains-mono-nerd` (Optional)
